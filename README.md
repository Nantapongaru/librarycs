# ระบบบริหารห้องสมุด

โปรเจกต์นี้เป็นระบบบริหารห้องสมุดที่พัฒนาด้วย PHP, MySQL และ Bootstrap สำหรับช่วยในการจัดการข้อมูลหนังสือ สมาชิก การยืมคืนหนังสือ ค่าปรับ และรายงานการใช้งาน

## คุณสมบัติหลัก

- จัดการข้อมูลหนังสือ
  - เพิ่มหนังสือ
  - แก้ไขข้อมูลหนังสือ
  - ลบหนังสือ
  - ตรวจสอบสถานะหนังสือว่าเป็นยืมได้หรือถูกยืมแล้ว
- จัดการสมาชิกและแผนก
  - เพิ่มและแก้ไขข้อมูลสมาชิก
  - จัดการข้อมูลแผนก
- ระบบยืม-คืนหนังสือ
  - บันทึกการยืมหนังสือ
  - บันทึกการคืนหนังสือ
  - คิดค่าปรับตามจำนวนวันที่เลยกำหนด
- แสดงสถิติและรายงาน
  - แสดงหนังสือที่ถูกยืมบ่อย
  - แสดงข้อมูลการยืมคืนในรูปแบบที่เข้าใจง่าย

## เทคโนโลยีที่ใช้

- PHP
- MySQL
- Bootstrap 5
- JavaScript
- HTML/CSS

## ความต้องการของระบบ

- XAMPP / WAMP / Laragon
- PHP 7 ขึ้นไป
- MySQL Database
- เว็บเบราว์เซอร์ เช่น Chrome, Edge, Firefox

## วิธีติดตั้งและรันโปรเจกต์

1. คัดลอกโฟลเดอร์โปรเจกต์ลงในไดเรกทอรีเว็บเซิร์ฟเวอร์ของคุณ
   - ถ้าใช้ XAMPP ให้วางไว้ที่ htdocs
2. เปิดเซิร์ฟเวอร์ Apache และ MySQL
3. สร้างฐานข้อมูล MySQL ตามชื่อที่โปรเจกต์ใช้งาน
   - ในไฟล์ connect.php มีการใช้ฐานข้อมูลชื่อ librarycs
4. แก้ไขรายละเอียดการเชื่อมต่อฐานข้อมูลในไฟล์ connect.php หากจำเป็น
5. เปิดเบราว์เซอร์และเข้าที่ URL ต่อไปนี้
   - http://localhost/libraeycs/index.php

## โครงสร้างไฟล์สำคัญ

- index.php : หน้าแรกของเว็บไซต์
- login.php : หน้าเข้าสู่ระบบ
- admin_book.php : หน้าจัดการหนังสือสำหรับผู้ดูแลระบบ
- borrow.php : หน้าทำรายการยืมหนังสือ
- return.php : หน้าทำรายการคืนหนังสือ
- connect.php : ไฟล์เชื่อมต่อฐานข้อมูล
- header.php / footer.php : ส่วนหัวและส่วนท้ายของหน้าเว็บ

## หมายเหตุ

โปรเจกต์นี้เหมาะสำหรับใช้เป็นผลงานหรือโปรเจกต์ฝึกปฏิบัติด้านเว็บแอปพลิเคชันและระบบจัดการข้อมูลในห้องสมุด โดยสามารถพัฒนาต่อได้ในด้านความปลอดภัย เช่น การเข้ารหัสรหัสผ่าน การตรวจสอบสิทธิ์ผู้ใช้ และการป้องกัน SQL Injection ให้ดีขึ้น

## ผู้พัฒนา

โปรเจกต์นี้พัฒนาขึ้นเพื่อเป็นระบบบริหารห้องสมุดสำหรับการศึกษาและการใช้งานในระดับพื้นฐาน
